## What is it?
YandexHub is a video hosting service that allows users to upload, view, rate, comment, save, and share videos.

## What used?
* Python 3.7+
* Django 3
* Django Rest Framework
* Django Channels
* JavaScript ES9
* JS Fetch Api
* CSS 3
* HTML 5
* Bootstrap 4

## How to run?
In order to run this project you need:
* Python 3.7+
