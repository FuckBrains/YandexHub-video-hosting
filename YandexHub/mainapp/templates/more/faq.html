When registering, there are several criteria:
  • The username must consist of at least one character and can not consist of spaces
  • The password must be at least 8 characters long, and consist of lowercase and uppercase letters of the Latin alphabet
  • The password cannot be the same as the username or email address

    Yandex Hub has a large number of restrictions on the number of videos, articles, and articles published per day.

    Videos
    Maximum of 5 videos per day.

    Tracks
    Maximum of 5 tracks per day.

    Articles
    Maximum of 10 articles per day.

    Comments
    Maximum of 10 comments under one video per day.

    Reply comments
    Maximum of 10 replies under one comment per day.

    Views
    Maximum of 10 views per video per day.

    Authentication

    YandexHub Public API uses HTTP Basic Access Authentication to ensure only correctly permissioned users are given access to the feeds they are allowed to read.

    Each API call made to YandexHub Public API is authenticated with a key generated on a per-user basis. YandexHub will provide you with an API key.

    This key is included in the HTTP header in the call made as follows:

    Authorization: Token [INSERT API KEY]
