YandexHub

The YandexHab project is my largest at the time of May 2021. The YandexHab project was not originally planned as a project in Yandex Lyceum in February 2021 I decided that my portfolio lacked some large and complex project without thinking twice I decided that I would write video hosting (a clone of YouTube).

After a while, I started writing a website layout on Bootstrap. My plan was that my friend and I would implement this big project together, I would write the back-end and he would write the front-end, but my friend started preparing for exams, so I had to write the entire front-end on my own. (The entire visual of the site is written very crookedly this can be understood simply by looking at the mobile version of the site that does not exist).

After I found out that my friend would not be able to write the front-end of the project, I decided to drop it, but after a while I found out about the project contest in Yandex Lyceum and decided that YouTube would be a great project for the contest. The YandexHub project is not a complete copy of YouTube unlike it YandexHub has its own telegram bot for notifications and on every page with a video or movie there is an online chat that allows viewers to discuss the video clip while watching it.

The site is located at yandexhub.ru.
